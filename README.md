# Rusterminal - v0.1.35
My old project, PYShell, but rewritten in Rust.

# Tested on:

- Arch Linux
- Linux Mint

# Install:

## 1. Clone the repo:

`git clone https://github.com/Meme-Supplier/Rusterminal.git`

## 2. Navigate to the source

`cd ~/Rusterminal/src`

## 3. Install Rusterminal

`bash install.sh`
